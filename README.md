# web-p2-reactcraft
## 실행 방법
``` shell
cd reactcraft/frontend
npm install
npm run build
npx http-server -c-1 -p <원하는 포트번호> -a localhost
```
🖥️ `localhost:<원하는 포트번호>`로 접속
